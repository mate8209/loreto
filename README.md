# Loreto, territorio posible

Presentación web para GitHub Pages del Trabajo Final 2026 del Doctorado en Desarrollo Territorial — UNRC.

## Tema
Proyecto preliminar de ordenamiento territorial participativo para Loreto, Corrientes.

## Integrantes
Juan Pablo Ybarra y Sebastián Slobayen.

## Uso
Subir todos los archivos de esta carpeta a la raíz del repositorio de GitHub y activar GitHub Pages desde:

Settings → Pages → Deploy from branch → main → /root

El archivo principal debe quedar como `index.html` en la raíz del repositorio.

## Navegación
Usar flechas del teclado o los botones inferiores. El guion oral se abre desde el botón `Guion`.
