## Cómo subir a GitHub Pages

1. Descomprimir el ZIP.
2. Entrar a la carpeta `loreto_doctoral_github`.
3. Subir **todos los archivos de adentro** a la raíz del repositorio.
4. Verificar que `index.html` quede en la raíz.
5. En GitHub: `Settings → Pages → Deploy from branch → main → /root`.
6. Esperar 1–2 minutos y abrir la URL de GitHub Pages.

La raíz del repositorio debe quedar así:

```text
index.html
styles.css
script.js
README.md
assets/
docs/
```

## Cómo usar la presentación

- Flecha derecha / barra espaciadora: avanzar.
- Flecha izquierda: retroceder.
- Botón “Guion oral”: abre el guion completo.
- Para imprimir o exportar a PDF: abrir en navegador y usar `Ctrl + P`.

## Enfoque académico

La presentación está formulada como **proyecto preliminar de ordenamiento territorial**, no como ejecución completa de un plan. Integra:

- Ciencia Transformadora.
- Inteligencia Territorial.
- Enfoque bottom-up.
- Metáfora Araña-Hormiga.
- Matriz FOAR.
- Gantt de seis meses.
- Factibilidad y posibilidad de concreción.
